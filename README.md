# ATProto Snapshotter

The goal is to be able to create a periodic snapshot for analytical purposes of the Bluesky Social Graph and Content. 

Included are
- Actors
- Posts
- Follows
- Likes
- Blocks

Excluded are:
- Lists
- Feeds + Feed Generators
- Threadgates

Only text content is stored locally, no media. It filters based on language and labelers. Defaults are english and the bsky default moderation.

For takedowns raise an issue on the GH page.
